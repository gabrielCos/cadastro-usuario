Projeto de cadastro de pessoas com apenas com nome e idade. 

Todos os usuários cadastrados serão mostrados abaixo do formulário para adicionar usuários.

Caso dados inváldos sejam colocados no formulário, um pop up é mostrado. Esse pop up pode ser descartado ao clicar no botão "okay" ou qualquer outro lugar da tela.
